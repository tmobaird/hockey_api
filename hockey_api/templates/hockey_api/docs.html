The Hockey API + Insomnia

Using an easy-to-use API client makes the process of testing, debugging, and developing APIs much more smooth. Insomnia is one of the best in this department. Because of this, The Hockey API provides an Insomnia configuration file for all users. Simply click the Download button below, download the json configuration, then in Insomnia click "Import" and import the json file. This will create a new collection in your Insomnia with all the requests you will need for The Hockey API. Enjoy!
